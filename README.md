# Linked Data - A Hands-on Introduction

A 4-hour Slidev workshop covering Linked Data, RDF, ontologies, SPARQL, and SHACL - with hands-on exercises using schema.org, Wikidata, and online validators.

## Usage

```bash
npm install
npm run dev      # dev server at http://localhost:3030
npm run build    # production build
npm run export   # export to PDF
```

Edit [slides.md](./slides.md) to modify the content.
